# Issue 2: 访谈计划与模板关联关系显示不正确
- **优先级**: P1
- **现象**: 计划列表中任意模板都能跳转到所有访谈计划，没有实际关联关系显示
- **根因**: 计划列表页面未显示模板名称，或链接未过滤
- **修复**: 在计划列表中显示关联模板名称，并可筛选

# Issue 3: 批量访谈统计分析功能
- **优先级**: P1
- **需求**: 针对一个访谈模板的全部访谈报告做统计分析，包括：
  - 共性问题挖掘（从千份访谈中提取高频主题）
  - 个性化有价值信息（特殊观点、创新建议）
  - 手工触发，非自动
- **实现**: 新增 `/admin/analysis/:templateId` 页面 + 触发 API
