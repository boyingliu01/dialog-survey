## OVERVIEW
Custom imperative state machine (NOT LangGraph StateGraph API). Manages interview lifecycle: planning → interviewing → analyzing → completed.


## CONVENTIONS
- **Node signature**: `async function nodeName(state, input?): Promise<Partial<InterviewState> & NodeOutput>`
- **State mutation**: Nodes return partial updates, merged via `{ ...state, ...result }`
- **No LangGraph channels**: State passed explicitly, not through channel system
- **Zod-first types**: `InterviewStateSchema` → `z.infer` for TypeScript type
- **NodeOutput**: `{ response?, shouldContinue?, nextQuestion? }` controls flow
